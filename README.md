# Sales Data Analysis using Python

This Week 2 internship project focuses on analyzing structured sales data using
Python to calculate revenue and generate business insights.

## Key Work Done
- Created a structured sales dataset
- Calculated total revenue and order count
- Performed product-wise and category-wise analysis
- Analyzed monthly sales trends
- Exported final dataset as CSV

## Tools Used
- Python
- Pandas
- Google Colab
- GitHub
